# 成员申请通道 / Membership Request Channel

本仓库用于处理笔记贡献组织的成员资格申请。欢迎对笔记创作、知识整理感兴趣的同学加入我们。

This repository handles membership requests for the Notes Contribution Organization. We welcome anyone interested in note-taking and knowledge sharing to join us.

---

## 📋 申请流程 / Application Process

### 1. 提交申请 / Submit an Application

点击上方 **Issues** 选项卡，然后点击 **New Issue**，选择 **"申请加入组织 / Organization Membership Request"** 模板，填写并提交。

Click the **Issues** tab above, then click **New Issue**, select the **"申请加入组织 / Organization Membership Request"** template, fill it out, and submit.

### 2. 等待审核 / Wait for Review

管理员会在 **3-5 个工作日** 内审核你的申请。如有需要补充的信息，管理员会在 Issue 评论区与你沟通。

Reviewers will respond within **3-5 business days**. If additional information is needed, we will reach out via issue comments.

### 3. 接收邀请 / Receive Invitation

审核通过后，你将收到来自 GitHub 的组织加入邀请邮件。请及时查收并点击 **Accept** 接受邀请。

Once approved, you will receive an organization invitation email from GitHub. Please check your inbox and click **Accept** to join.

---

## 📝 申请要求 / Requirements

| 项目 / Item | 说明 / Description |
| :--- | :--- |
| **真实姓名 / Full Name** | 请填写你的真实姓名 |
| **学校邮箱 / School Email** | 仅限 `@westlake.edu.cn` 邮箱 |
| **联系电话 / Phone Number** | 用于紧急情况联系 |
| **申请理由 / Motivation** | 简要说明加入原因及兴趣方向 |
